# Jogo de Sobrevivência 2D (Pygame)

Este é um jogo de sobrevivência desenvolvido em Python utilizando a biblioteca Pygame. O projeto foi criado como atividade prática para a faculdade UNINTER, tendo recebido **Nota Maxima (100)**.

## 🚀 Funcionalidades
- Menu inicial interativo com instruções de comandos.
- Sistema de dificuldade selecionável (Fácil / Difícil).
- Sistema de ondas/hordas de inimigos.
- Controles de áudio integrados (Aumentar, diminuir e mutar o volume).
- Tratamento de transparência nos assets visuais.

## 🎮 Controles
- **Movimentação:** Teclas `W`, `A`, `S`, `D`
- **Atirar:** Barra de `ESPAÇO`
- **Pausar o jogo:** Tecla `P`
- **Controle de Volume:** Teclas `+` e `-`
- **Mutar áudio:** Tecla `M`
- **Dificuldade (no menu):** Tecla `1` (Fácil) ou Tecla `2` (Difícil)

## 🛠️ Como rodar o projeto
1. Certifique-se de ter o Python instalado na sua máquina.
2. Instale a biblioteca Pygame executando: `pip install pygame`
3. Baixe os arquivos deste repositório.
4. Execute o arquivo principal: `python main.py`
